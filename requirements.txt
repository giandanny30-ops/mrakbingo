====================================================
  SQUAD Discord Bot — Uputstvo za pokretanje
====================================================

KORAK 1 — Otvori fajl: bot.py
   Promijeni linije 6 i 7:

   TOKEN = "OVDJE_STAVI_BOT_TOKEN"
   CHANNEL_ID = 1234567890

   Zamijeni sa tvojim podacima, npr:
   TOKEN = "MTUyNjE2OTU2NjA1OTgyMzE2NQ.GAIMhk.xxx"
   CHANNEL_ID = 1526169748499468379

KORAK 2 — Instaliraj pakete (samo jednom):
   Pokreni u terminalu:
   pip install -r requirements.txt

   requirements.txt sadrzi SAMO ovo (ne mijenjaj!):
   discord.py==2.4.0
   audioop-lts==0.2.1

KORAK 3 — Pokreni bota:
   python bot.py

====================================================
FAJLOVI:
  bot.py            <- JEDINI fajl koji mijenjas (token i kanal)
  requirements.txt  <- NE MIJENJAJ, samo pokreni pip install
  games/            <- NE MIJENJAJ, logika igara
